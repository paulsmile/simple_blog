一个使用Django和Bootstrap搭建起来的博客    
此博客目前已经部署到SAE[（地址）](http://cjyfff.sinaapp.com)   
    
###依赖：    
django == 1.6    
python == 2.7   
markdown == 2.4    
JQuery == 1.11   
BootStrap == 2.2.2   
factory-boy == 2.4.1    
    
###目前已经实现的功能：    
1、博客正文支持markdown语法，支持代码高亮    
2、博客正文中的段落、图片代码等内容能够按照指定的顺序来显示    
3、支持评论功能，支持验证码(使用JS实现)    
4、首页实现了分页显示    
5、实现了RSS功能
