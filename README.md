一个使用Django和Bootstrap搭建起来的博客    
基本功能已经完善，后续有时间我将继续完善这个程序    
    
###依赖：    
1、django 1.6    
2、python 2.7   
3、django-contrib-comments    
4、markdown      
    
###目前已经实现的功能：    
1、博客正文支持markdown语法，支持代码高亮    
2、博客正文中的段落、图片代码等内容能够按照指定的顺序来显示    
3、支持评论功能，支持验证码(使用JS实现)    
4、首页实现了分页显示    
5、实现了RSS功能    

###目前存在的问题：
